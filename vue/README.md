# DevExtreme Vue NumberBox - Getting Started 

This repository stores the code examples of the Vue NumberBox component for the [Getting Started with NumberBox](https://js.devexpress.com/Documentation/Guide/UI_Components/NumberBox/Getting_Started_with_NumberBox/) tutorial. The component allows users to enter a value and increment or decrement it with the spin buttons, keyboard, or mouse wheel.

## Project setup
```
npm install
```
